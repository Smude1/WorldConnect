# Features Directory

The system is organized around several high-level features:

1. **User Authentication & Management**
   - Secure login & signup with JWT / Supabase.
   - Session verification and protected route middleware.

2. **User Profiles**
   - Full customized profiles with avatars, bios, and links.
   - Public view and editable personal settings.

3. **Explore & Search**
   - Discovery feed to connect with new users.
   - Query filters to find people by location, name, or interests.

4. **Realtime Chat / Conversations**
   - One-to-one and group messaging.
   - Typings indicators, delivery statuses, and read state.

5. **Notification System**
   - Push and in-app alerts for conversations and actions.
   - Mark as read/unread controls.
