# WorldConnect

WorldConnect is a web application with a Next.js frontend and an Express + Prisma backend.

## Structure
- `client/`: Next.js client application
- `server/`: Express backend application
- `docs/`: Product and API documentation

## Getting Started
To get started:
1. Run `npm install` at the root.
2. Fill in the environment files (`client/.env.local` and `server/.env`).
